# Frontend

React + Vite + TypeScript frontend placeholder.

Quick start:

- `cd frontend`
- `npm install`
- `npm run dev`

Configuration:

- Set `VITE_API_URL` to the backend API base URL (e.g. `http://localhost:8000`).

Developer focus:

- `frontend/src/index.tsx` — app entry point
- `frontend/src/modules/auth/login/index.tsx` — login flow module
- `frontend/src/modules/dashboard/index.tsx` — dashboard module
- `frontend/src/modules/reports/index.tsx` — report list module
- `frontend/src/modules/pipeline/index.tsx` — pipeline control module

Related OpenSpec contract:

- `openspec/specs/ui_and_login/ui_login_contract.md`
